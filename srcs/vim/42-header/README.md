42-toolkit	42 Header
==========

This plugin generate 42 header.

Auto dispatch:

    - ./configure

Or manual:

    - echo "source /path/to/make_header.vim" >> $HOME/.vimrc
    - echo "export MAIL42=YourMail" >> YourProfileFile
    - echo "export USER42=YourUser" >> YourProfileFile
