42-toolkit	vim
==========

## How to install ?

All plugins are copy to $HOME/.vim/

	install or update all plugin:
	- ./install

	individually:
    - cd PluginDirectory
    - ./configure

## Plugins

    - header		"Make header like 42"
    - generator-h	"Make .h like 42"
	- speed_open	"Switch between .c and .h"
	- syntax		"Highlight lib42 keyword"
