42-toolkit	vim
==========

## How to install

All plugins are copy to $HOME/.vim/plugin

    - cd PluginDirectory
    - ./configure

## Plugins

    - header		"Make header like 42"
    - generator-h	"Make .h like 42"
	- speed_open	"Switch between .c and .h"
