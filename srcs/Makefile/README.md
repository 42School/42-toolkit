42-toolkit	Makefile
==========

Rules:

    - clean		Cleaning all object
    - fclean	Cleaning all object and executable
    - debug		Add debug flags
    - re		Deleting all object and build

    - analyze	Analyze code but compile nothing
    - every		God code
    - help		verbose arguments
